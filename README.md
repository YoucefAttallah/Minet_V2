# Classification de minéraux – Adaptation de DenseNet-201

Ce projet reprend et modifie le notebook Kaggle ["Minerals F1 Score 90"](https://www.kaggle.com/code/gpiosenka/minerals-f1-score-90) de **gpiosenka**.

### Modifications apportées
- Optimisation des hyperparamètres et de la procédure d’entraînement.
- Sélection de **DenseNet-201** comme modèle final (meilleurs résultats).
- Tests sur d’autres architectures (dont RepVit, non retenu dans l’article).
